# 🚀 Payment Tracker App (Level 2)

## 📌 Description
A Stellar blockchain multi-wallet payment tracker built with React and Stellar SDK.

## ✨ Features
- Multi-wallet support
- Check balances
- Send XLM
- Transaction status
- 3 Error handling types

## ⚙️ Run Locally
npm install
npm run dev

## 📸 Screenshots
Add wallet.png, balance.png, success.png inside screenshots folder.
